Section 4: HTML Terms

Webpage-এর কাঠামো তৈরির ভাষা — HyperText Markup Language

▼
01

Tag

ট্যাগ ▼
📖
বাংলা অর্থ
HTML-এ কোনো কিছু শুরু ও শেষ বোঝানোর চিহ্ন — <> বন্ধনীর ভেতরে লেখা
💡
সহজ ব্যাখ্যা
Tag হলো HTML-এর নির্দেশ। Opening tag <p> মানে paragraph শুরু, Closing tag </p> মানে paragraph শেষ।
📄 উদাহরণ
<h1>এটি একটি শিরোনাম</h1>
<p>এটি একটি paragraph</p>
<img> ← self-closing tag, / লাগে না
02

Element

এলিমেন্ট ▼
📖
বাংলা অর্থ
Opening tag + Content + Closing tag — এই তিনটি মিলে একটি Element
💡
সহজ ব্যাখ্যা
Tag মানে শুধু চিহ্ন। Element মানে সম্পূর্ণ জিনিস। যেমন সিম্পল বললে: Tag = বাক্সের ঢাকনা, Element = ঢাকনা+বাক্স+ভেতরের জিনিস।
📄 উদাহরণ
← এটা একটা সম্পূর্ণ Element:
<p>আমি বাংলাদেশে থাকি</p>
  ↑          ↑              ↑
Opening  Content        Closing
03

Attribute

অ্যাট্রিবিউট ▼
📖
বাংলা অর্থ
HTML element-এর অতিরিক্ত তথ্য বা বৈশিষ্ট্য যা tag-এর ভেতরে লেখা হয়
💡
সহজ ব্যাখ্যা
মানুষের পরিচয়পত্রে নাম, বয়স, ঠিকানা থাকে — সেগুলোই attribute। Element-কে আরও বিস্তারিত বলে দেওয়া।
📄 উদাহরণ
<img src="photo.jpg" alt="আমার ছবি">
<a href="https://google.com">Google</a>
<input type="text" placeholder="নাম লিখুন">
04

Div

ডিভ / বিভাজন ▼
📖
বাংলা অর্থ
Division — একটি generic container যা অন্য element-দের গ্রুপ করে রাখে
💡
সহজ ব্যাখ্যা
Div হলো একটি অদৃশ্য বাক্স। তার ভেতরে যা খুশি রাখো। CSS দিয়ে সাজাও। HTML-এ সবচেয়ে বেশি ব্যবহৃত tag।
📄 উদাহরণ
<div class="card">
  <h2>পণ্যের নাম</h2>
  <p>দাম: ৫০০ টাকা</p>
</div>
05

Span

স্প্যান / ইনলাইন ▼
📖
বাংলা অর্থ
Div-এর মতো কিন্তু inline — নতুন line-এ যায় না, text-এর মাঝেই থাকে
💡
সহজ ব্যাখ্যা
Div = পুরো লাইন নেয়। Span = শুধু যতটুকু দরকার। Text-এর একটা অংশে আলাদা color বা style দিতে span।
📄 উদাহরণ
<p>দাম: <span style="color:red">৫০০ টাকা</span></p>
↑ "৫০০ টাকা" অংশটুকুই লাল হবে, বাকিটা না
06

Heading

হেডিং / শিরোনাম ▼
📖
বাংলা অর্থ
শিরোনাম — h1 থেকে h6 পর্যন্ত, h1 সবচেয়ে বড়, h6 সবচেয়ে ছোট
💡
সহজ ব্যাখ্যা
বইয়ে যেমন Chapter title, Sub-title, Section title থাকে — তেমনি webpage-এ h1, h2, h3 দিয়ে hierarchy বোঝানো হয়।
📄 উদাহরণ
<h1>সবচেয়ে বড় শিরোনাম</h1>   ← Page title
<h2>মাঝারি শিরোনাম</h2>       ← Section
<h3>ছোট শিরোনাম</h3>           ← Sub-section
07

Paragraph

প্যারাগ্রাফ ▼
📖
বাংলা অর্থ
সাধারণ body text লেখার জন্য <p> tag
💡
সহজ ব্যাখ্যা
কোনো article বা description লিখলে <p> ব্যবহার করো। Browser automatically উপরে-নিচে space দেয়।
📄 উদাহরণ
<p>বাংলাদেশ দক্ষিণ এশিয়ার একটি সুন্দর দেশ।
এখানে ১৭ কোটি মানুষ বাস করে।</p>
08

Link (Anchor)

লিংক / হাইপারলিংক ▼
📖
বাংলা অর্থ
Click করলে অন্য page বা section-এ নিয়ে যাওয়ার <a> tag
💡
সহজ ব্যাখ্যা
Internet-এর web = link-এর জালের কারণে। একটা page থেকে আরেকটায় যাওয়ার রাস্তাই Link। href-এ destination লেখো।
📄 উদাহরণ
<a href="https://google.com">Google যাও</a>
<a href="#section2">এই page-এ নামো</a>
<a href="tel:01700000000">Call করো</a>
09

Image

ছবি / ইমেজ ▼
📖
বাংলা অর্থ
Webpage-এ ছবি দেখানোর <img> tag
💡
সহজ ব্যাখ্যা
img tag self-closing — closing tag লাগে না। src-এ ছবির path, alt-এ ছবির description দাও। Alt text দৃষ্টিপ্রতিবন্ধীদের জন্য জরুরি।
📄 উদাহরণ
<img src="banner.jpg"
     alt="আমার website-এর banner"
     width="600">
10

Table

টেবিল / সারণি ▼
📖
বাংলা অর্থ
Row ও Column-এ data সাজিয়ে দেখানোর HTML structure
💡
সহজ ব্যাখ্যা
Excel-এর মতো ঘর-ঘর করা। <table> = পুরো table, <tr> = row, <td> = cell, <th> = header cell।
📄 উদাহরণ
<table>
  <tr><th>নাম</th><th>বয়স</th></tr>
  <tr><td>Aminul</td><td>25</td></tr>
</table>
11

Form

ফর্ম / HTML Form ▼
📖
বাংলা অর্থ
User-এর কাছ থেকে data নেওয়ার জন্য <form> tag ও তার ভেতরের input elements
💡
সহজ ব্যাখ্যা
Login করার সময় username-password যেখানে লেখো — সেটা HTML Form। Submit করলে server-এ data যায়।
📄 উদাহরণ
<form action="/login" method="POST">
  <input type="text"     placeholder="ইমেইল">
  <input type="password" placeholder="পাসওয়ার্ড">
  <button type="submit">Login</button>
</form>
🎨

Section 5: CSS Terms

HTML-কে সুন্দর করার ভাষা — Cascading Style Sheets

▼
01

Color

রঙ ▼
📖
বাংলা অর্থ
Text বা element-এর রঙ নির্ধারণ করার CSS property
💡
সহজ ব্যাখ্যা
CSS-এ রঙ তিনভাবে দেওয়া যায়: Name (red, blue), HEX (#ff0000), RGB (rgb(255,0,0))।
🎨 উদাহরণ
h1 { color: red; }
p  { color: #4f46e5; }  ← HEX
.title { color: rgb(255, 100, 0); }
02

Background

ব্যাকগ্রাউন্ড / পটভূমি ▼
📖
বাংলা অর্থ
Element-এর পেছনের রঙ বা ছবি সেট করা
💡
সহজ ব্যাখ্যা
ঘরের দেয়ালের রঙ বা wallpaper = Background। Solid color, gradient, বা image হতে পারে।
🎨 উদাহরণ
body { background-color: #f0f2fa; }
.hero {
  background: linear-gradient(135deg, #4f46e5, #7c3aed);
}
.banner {
  background-image: url('bg.jpg');
}
03

Margin

মার্জিন / বাইরের ফাঁকা ▼
📖
বাংলা অর্থ
Element-এর বাইরের ফাঁকা জায়গা — অন্য element থেকে দূরত্ব
💡
সহজ ব্যাখ্যা
বাড়ি থেকে রাস্তার দূরত্ব = Margin। দুটো element-এর মাঝখানের ফাঁকা space।
🎨 উদাহরণ
.card {
  margin: 20px;          ← চারদিকে 20px
  margin-top: 10px;      ← শুধু উপরে
  margin: 10px 20px;    ← উপর-নিচ, বাম-ডান
  margin: auto;          ← মাঝখানে center করে
}
04

Padding

প্যাডিং / ভেতরের ফাঁকা ▼
📖
বাংলা অর্থ
Element-এর ভেতরের ফাঁকা জায়গা — content ও border-এর মধ্যে
💡
সহজ ব্যাখ্যা
বাক্সের ভেতরে foam বা bubble wrap = Padding। Content-কে border থেকে দূরে রাখে।
মনে রাখো: Margin = বাইরে, Padding = ভেতরে।
🎨 উদাহরণ
button {
  padding: 12px 24px;  ← উপর-নিচ 12, বাম-ডান 24
}
.card {
  padding: 20px;       ← চারদিকে 20px breathing room
}
05

Border

বর্ডার / সীমানা ▼
📖
বাংলা অর্থ
Element-এর চারদিকের রেখা বা সীমানা
💡
সহজ ব্যাখ্যা
বাড়ির চারদিকের দেয়াল বা fence = Border। Thickness, style (solid/dashed), color সব set করা যায়।
🎨 উদাহরণ
.box {
  border: 2px solid #4f46e5;
  border-radius: 10px;     ← গোলাকার কোণ
  border-left: 4px solid red; ← শুধু বাম দিকে
}
06

Display

ডিসপ্লে / প্রদর্শন ▼
📖
বাংলা অর্থ
Element কীভাবে layout-এ জায়গা নেবে সেটা নির্ধারণ করা
💡
সহজ ব্যাখ্যা
block = পুরো line নেয় (div, p, h1)। inline = শুধু প্রয়োজনমতো (span, a)। none = লুকিয়ে ফেলে।
🎨 উদাহরণ
.hide   { display: none; }      ← অদৃশ্য
.show   { display: block; }     ← পুরো line
.inline { display: inline; }    ← ভেতরে ভেতরে
.flex   { display: flex; }      ← flexbox layout
07

Flexbox

ফ্লেক্সবক্স ▼
📖
বাংলা অর্থ
Elements-কে row বা column-এ সহজে সাজানোর CSS layout system
💡
সহজ ব্যাখ্যা
Flexbox = smart organizer। সে নিজেই সব elements-কে সমান ভাগ করে, মাঝখানে রাখে, ফাঁক দেয়। কঠিন layout সহজ হয়ে যায়।
🎨 উদাহরণ
.nav {
  display: flex;
  justify-content: space-between; ← দুই পাশে
  align-items: center;           ← মাঝখানে উপর-নিচ
  gap: 16px;                       ← মাঝে ফাঁক
}
08

Grid

গ্রিড / জাল বিন্যাস ▼
📖
বাংলা অর্থ
Row ও Column দুটো দিক নিয়ন্ত্রণ করে layout বানানোর CSS system
💡
সহজ ব্যাখ্যা
Flexbox = একদিকে (row বা column)। Grid = দুই দিকেই। Chess board-এর মতো ঘর-ঘর করে layout।
🎨 উদাহরণ
.product-list {
  display: grid;
  grid-template-columns: repeat(3, 1fr); ← ৩ কলাম
  gap: 20px;
}
09

Position

পজিশন / অবস্থান ▼
📖
বাংলা অর্থ
Element ঠিক কোথায় থাকবে সেটা নির্ধারণ করা
💡
সহজ ব্যাখ্যা
static = স্বাভাবিক। relative = নিজের জায়গা থেকে সরে। absolute = parent-এর মধ্যে fixed। sticky = scroll করলেও আটকে থাকে।
🎨 উদাহরণ
.topnav {
  position: sticky;  ← scroll করলেও উপরে থাকে
  top: 0;
}
.badge {
  position: absolute;
  top: -8px; right: -8px;  ← কোণায় বসানো
}
10

Animation

অ্যানিমেশন ▼
📖
বাংলা অর্থ
CSS দিয়ে element-কে নড়াচড়া করানো বা রূপান্তর করানো
💡
সহজ ব্যাখ্যা
@keyframes দিয়ে শুরু থেকে শেষের অবস্থা বলো, CSS বাকি সব নিজে করবে। Loading spinner, fade in, slide — সব animation।
🎨 উদাহরণ
@keyframes spin {
  from { transform: rotate(0deg); }
  to   { transform: rotate(360deg); }
}
.spinner {
  animation: spin 1s linear infinite;
}
11

Transition

ট্রানজিশন / মসৃণ পরিবর্তন ▼
📖
বাংলা অর্থ
CSS-এর কোনো property পরিবর্তনকে ধীরে ধীরে smooth করে দেখানো
💡
সহজ ব্যাখ্যা
Button hover করলে হঠাৎ রঙ না বদলে আস্তে আস্তে বদলানো = Transition। Animation-এর মতো কিন্তু সহজ। শুধু একটা লাইন লাগে।
🎨 উদাহরণ
button {
  background: blue;
  transition: background 0.3s ease;
}
button:hover {
  background: purple;  ← আস্তে আস্তে বদলাবে
}
12

Responsive Design

রেসপন্সিভ ডিজাইন ▼
📖
বাংলা অর্থ
Website যে screen-এ দেখা যাচ্ছে তার size অনুযায়ী নিজেকে মানিয়ে নেওয়া
💡
সহজ ব্যাখ্যা
Mobile, Tablet, Desktop — তিন ধরনের screen-এ একই website সুন্দর দেখা = Responsive। @media query দিয়ে করা হয়।
🎨 উদাহরণ
/* Desktop: 3 column */
.grid { grid-template-columns: repeat(3, 1fr); }

/* Mobile: 1 column */
@media (max-width: 480px) {
  .grid { grid-template-columns: 1fr; }
}
⚡

Section 6: JavaScript Terms

Webpage-কে জীবন্ত ও interactive করার ভাষা

▼
01

Variable (let / const / var)

ভেরিয়েবল – JS ▼
📖
বাংলা অর্থ
JavaScript-এ data রাখার জায়গা। তিনটি keyword: let, const, var
💡
পার্থক্য
let = পরিবর্তনযোগ্য, নতুন style। const = পরিবর্তনহীন। var = পুরনো style, এখন কম ব্যবহার হয়।
⚡ উদাহরণ
let   name  = "Aminul";    ← পরে বদলানো যাবে
const PI    = 3.14159;    ← বদলানো যাবে না
let   count = 0;
count = count + 1;          ← এখন count = 1
02

Function

ফাংশন – JS ▼
📖
বাংলা অর্থ
বারবার ব্যবহারযোগ্য code-এর block। Traditional ও Arrow function দুটোই আছে।
💡
সহজ ব্যাখ্যা
Arrow function (=>) হলো modern shortcut। একই কাজ কম লেখায়। আজকের JS-এ Arrow function বেশি ব্যবহার হয়।
⚡ উদাহরণ
// Traditional function
function greet(name) {
  return "হ্যালো, " + name;
}

// Arrow function (modern)
const greet = (name) => "হ্যালো, " + name;

greet("Aminul"); ← "হ্যালো, Aminul"
03

Event & Listener

ইভেন্ট ও লিসেনার ▼
📖
বাংলা অর্থ
Event = user-এর action (click, scroll, keypress)। Listener = সেই action-এর জন্য অপেক্ষারত code।
💡
সহজ ব্যাখ্যা
দারোয়ান সবসময় দরজায় থাকে, কেউ knock করলে ভেতরে জানায় = Event Listener। JavaScript সবসময় "শুনছে", event হলে কাজ করে।
⚡ উদাহরণ
const btn = document.getElementById('myBtn');

btn.addEventListener('click', () => {
  alert('Button-এ click হয়েছে!');
});

// অন্য events: 'scroll', 'keydown', 'submit'

DOM Manipulation

ডম ম্যানিপুলেশন ▼

বাংলা অর্থ
JavaScript দিয়ে HTML element খোঁজা, পড়া, পরিবর্তন করা

সহজ ব্যাখ্যা
HTML লেখার পরে JS দিয়ে সেই HTML-এর যেকোনো অংশ real-time-এ বদলানো যায় — text, color, class, visibility সব।

// Element খোঁজা
const title = document.getElementById('title');
const cards = document.querySelectorAll('.card');

// পরিবর্তন করা
title.textContent = 'নতুন শিরোনাম';

title.classList.add('active');
05

Fetch API

ফেচ এপিআই ▼
📖
বাংলা অর্থ
JavaScript দিয়ে server বা external API থেকে data আনার modern পদ্ধতি
💡
সহজ ব্যাখ্যা
Page reload না করেই background-এ server থেকে নতুন data আনা = Fetch। Facebook-এ scroll করলে নতুন post আসে — এটাই Fetch API।
⚡ উদাহরণ
async function getData() {
  const response = await fetch('https://api.example.com/users');
  const data = await response.json();
  console.log(data); ← data এসে গেছে!
}
getData();
06

LocalStorage

লোকাল স্টোরেজ ▼
📖
বাংলা অর্থ
Browser-এ permanently data save করার জায়গা — server ছাড়াই
💡
সহজ ব্যাখ্যা
তোমার phone-এর টুকিটাকি settings মনে রাখার মতো। Page বন্ধ করলেও, Browser বন্ধ করলেও data থাকে। Server দরকার নেই।
⚡ উদাহরণ
// Save করা
localStorage.setItem('theme', 'dark');
localStorage.setItem('name', 'Aminul');

// পড়া
const theme = localStorage.getItem('theme');

// মুছে ফেলা
localStorage.removeItem('theme');
🌍 বাস্তব ব্যবহার

এই page-এর Dark/Light mode choice ও তোমার "পড়া হয়েছে" progress — সবই LocalStorage-এ save হচ্ছে!

07

SessionStorage

সেশন স্টোরেজ ▼
📖
বাংলা অর্থ
LocalStorage-এর মতো, কিন্তু Browser tab বন্ধ করলে data মুছে যায়
💡
পার্থক্য
LocalStorage = চিরকাল থাকে (না মুছলে)। SessionStorage = tab বন্ধ মানেই শেষ। যেমন shopping cart preview।
⚡ উদাহরণ
// একই syntax, কিন্তু tab বন্ধে মুছে যায়
sessionStorage.setItem('tempData', '12345');
const temp = sessionStorage.getItem('tempData');

// ব্যবহার: wizard form-এর step data,
// one-time login session info
🔍

কোনো result পাওয়া যায়নি।
