Section 1: Programming Basics

প্রোগ্রামিংয়ের মূল ধারণাগুলো সহজ বাংলায়

▼
01

Programming

প্রোগ্রামিং ▼
📖
বাংলা অর্থ
কম্পিউটারকে নির্দেশ দেওয়ার প্রক্রিয়া
💡
সহজ ব্যাখ্যা
কম্পিউটার নিজে থেকে কিছু করতে পারে না। তাকে step-by-step নির্দেশ দিতে হয়। এই নির্দেশ লেখার কাজটাই হলো Programming।
🌍 বাস্তব উদাহরণ

তুমি রান্না করার Recipe লিখলে — কতটুকু লবণ, কখন নামাবে। এটাই Programming-এর মতো। Recipe = Program, রান্না = Computer-এর কাজ।

02

Coding

কোডিং ▼
📖
বাংলা অর্থ
Code লেখার কাজ — Programming-এর practical অংশ
💡
সহজ ব্যাখ্যা
Programming হলো বড় concept, আর Coding হলো সেই concept অনুযায়ী code লেখা। যেমন — HTML, CSS, JavaScript লেখা = Coding।
🌍 বাস্তব উদাহরণ

একটা ঘর বানানোর plan করা = Programming। সেই plan অনুযায়ী ইট-সিমেন্ট দিয়ে দেয়াল তোলা = Coding।

03

Software

সফটওয়্যার ▼
📖
বাংলা অর্থ
Code দিয়ে তৈরি প্রোগ্রাম যা computer-এ চলে
💡
সহজ ব্যাখ্যা
Hardware হলো computer-এর শরীর (CPU, RAM, Screen), আর Software হলো সেই শরীরে যে প্রাণ থাকে — অর্থাৎ program ও apps।
🌍 বাস্তব উদাহরণ

Microsoft Word, Google Chrome, WhatsApp — এগুলো সবই Software। তুমি এগুলো দেখতে পাও কিন্তু ধরতে পারো না — শুধু screen-এ ব্যবহার করো।

04

Application

অ্যাপ্লিকেশন / অ্যাপ ▼
📖
বাংলা অর্থ
একটি নির্দিষ্ট কাজের জন্য তৈরি Software
💡
সহজ ব্যাখ্যা
Software একটি বড় category। Application হলো তার ভেতরে যেটা user-দের কোনো specific কাজে সাহায্য করে। Mobile App, Desktop App, Web App — সব Application।
🌍 বাস্তব উদাহরণ

Facebook App = Social media application। Bkash App = Banking application। VS Code = Code editor application।

05

Website

ওয়েবসাইট ▼
📖
বাংলা অর্থ
Internet-এ থাকা একটি address-এ যাওয়া যায় এমন pages-এর সমষ্টি
💡
সহজ ব্যাখ্যা
Internet-এ একটি ঠিকানা (URL) দিয়ে যে page খোলে, সেটাই Website। সাধারণত Information দেখানোর জন্য ব্যবহার হয়।
🌍 বাস্তব উদাহরণ

তোমার স্কুলের website, কোনো newspaper-এর website যেখানে শুধু পড়তে পারো — interact কম। যেমন: prothomalo.com

06

Web App

ওয়েব অ্যাপ ▼
📖
বাংলা অর্থ
Browser-এ চলে এমন interactive Application
💡
সহজ ব্যাখ্যা
Website শুধু দেখায়, কিন্তু Web App-এ তুমি কাজ করতে পারো — Login করা, Data save করা, Message পাঠানো ইত্যাদি।
🌍 বাস্তব উদাহরণ

Gmail, Google Docs, Facebook — এগুলো Web App। Browser খুলে ব্যবহার করো, install করা লাগে না।

07

Browser

ব্রাউজার ▼
📖
বাংলা অর্থ
Internet-এ website/app দেখার software
💡
সহজ ব্যাখ্যা
Browser হলো একটা window — যার মাধ্যমে তুমি internet-এর সব website দেখতে পাও। HTML, CSS, JS পড়ে সেটাকে সুন্দর করে screen-এ দেখায়।
🌍 বাস্তব উদাহরণ

Google Chrome, Mozilla Firefox, Microsoft Edge, Samsung Internet — এগুলো সব Browser।

08

Server

সার্ভার ▼
📖
বাংলা অর্থ
যে computer সবসময় চালু থেকে অন্যদের data ও service দেয়
💡
সহজ ব্যাখ্যা
তুমি browser-এ facebook.com লিখলে — Facebook-এর Server তোমার কাছে সেই page পাঠিয়ে দেয়। Server = service দেওয়া computer।
🌍 বাস্তব উদাহরণ

একটা restaurant-এ waiter যেমন খাবার এনে দেয়, Server ঠিক তেমনি data এনে দেয়। তুমি = Client, Restaurant = Server।

09

Database

ডেটাবেস ▼
📖
বাংলা অর্থ
সুশৃঙ্খলভাবে data সংরক্ষণ করার system
💡
সহজ ব্যাখ্যা
সব user-দের নাম, password, post — এগুলো কোথাও save থাকতে হয়। সেই জায়গাটার নাম Database। Excel spreadsheet-এর মতো, কিন্তু অনেক বড় ও শক্তিশালী।
🌍 বাস্তব উদাহরণ

তোমার স্কুলের সব ছাত্রছাত্রীর তথ্য একটা register book-এ থাকে — সেটাই Database-এর real-life উদাহরণ।

10

API

এপিআই ▼
📖
বাংলা অর্থ
Application Programming Interface — দুটো software-এর মধ্যে যোগাযোগের নিয়ম
💡
সহজ ব্যাখ্যা
API হলো একটা bridge বা দরজা। একটা app অন্য app-এর কাছ থেকে data নিতে API ব্যবহার করে। তুমি সরাসরি ঢুকতে পারবে না, API-র মাধ্যমে চাইতে হবে।
🌍 বাস্তব উদাহরণ

Pathao app Google Maps দেখায় — কারণ Pathao, Google Maps-এর API ব্যবহার করে। Restaurant = Google Maps, Waiter = API, তুমি = Pathao App।

11

Framework

ফ্রেমওয়ার্ক ▼
📖
বাংলা অর্থ
একটি pre-built কাঠামো যা দ্রুত app বানাতে সাহায্য করে
💡
সহজ ব্যাখ্যা
Framework হলো বাড়ির ready-made ছাঁচ বা mold। তুমি সেই ছাঁচে ঢালাই করো, পুরোটা নিজে বানাতে হয় না। এতে সময় বাঁচে।
🌍 বাস্তব উদাহরণ

React (JavaScript), Bootstrap (CSS), Django (Python) — এগুলো সব Framework। Bootstrap দিয়ে button বানাতে ১ লাইন, নিজে বানাতে ২০ লাইন লাগতো।

12

Library

লাইব্রেরি ▼
📖
বাংলা অর্থ
Ready-made code-এর সংগ্রহ যা নিজের project-এ ব্যবহার করা যায়
💡
সহজ ব্যাখ্যা
Library = বইয়ের লাইব্রেরি। অনেক বই সাজানো আছে, দরকার মতো নিয়ে পড়। Framework-এর চেয়ে ছোট, শুধু specific কাজের জন্য।
🌍 বাস্তব উদাহরণ

Chart.js = chart বানানোর library। date-fns = date handle করার library। তুমি শুধু call করো, ওরা কাজ করে দেয়।

13

SDK

এসডিকে ▼
📖
বাংলা অর্থ
Software Development Kit — একটি platform-এর জন্য সম্পূর্ণ developer toolkit
💡
সহজ ব্যাখ্যা
SDK হলো একটি complete toolbox। যেমন Android SDK-তে Android app বানানোর সব tools, libraries, emulator একসাথে পাবে।
🌍 বাস্তব উদাহরণ

Android app বানাতে Android SDK লাগে। Facebook app-এ login button দিতে Facebook SDK লাগে।

14

IDE

কোড লেখার সফটওয়্যার ▼
📖
বাংলা অর্থ
Integrated Development Environment — code লেখার smart software
💡
সহজ ব্যাখ্যা
IDE হলো coder-এর Microsoft Word। সাধারণ text editor-এর চেয়ে বেশি সুবিধা — auto-complete, error highlight, run করার button, সব একসাথে।
🌍 বাস্তব উদাহরণ

VS Code (সবচেয়ে popular), Android Studio (Android-এর জন্য), PyCharm (Python-এর জন্য) — সব IDE।

15

Compiler

কম্পাইলার ▼
📖
বাংলা অর্থ
Human-readable code কে computer-readable machine code-এ রূপান্তর করে
💡
সহজ ব্যাখ্যা
তুমি English-এ code লেখো, computer বোঝে শুধু 0 আর 1। Compiler হলো translator — সে English code কে 0-1 language-এ translate করে।
🌍 বাস্তব উদাহরণ

C, C++, Java — এগুলো Compiled language। তুমি code লেখো, Compiler পুরো code একবারে translate করে, তারপর run হয়।

16

Interpreter

ইন্টারপ্রেটার ▼
📖
বাংলা অর্থ
Code line-by-line পড়ে সাথে সাথে execute করে
💡
সহজ ব্যাখ্যা
Compiler একসাথে পুরো বই translate করে। Interpreter হলো live translator — একটা বাক্য পড়ো, সাথে সাথে translate করো।
🌍 বাস্তব উদাহরণ

Python, JavaScript — এগুলো Interpreted language। Browser JavaScript interpret করে চালায়। তাই instantly দেখা যায়।

17

Algorithm

অ্যালগরিদম ▼
📖
বাংলা অর্থ
কোনো সমস্যা সমাধানের step-by-step পরিকল্পনা
💡
সহজ ব্যাখ্যা
রান্নার recipe, ঘর থেকে school যাওয়ার রাস্তা — সবকিছুই Algorithm। একটা লক্ষ্য অর্জনের জন্য নির্দিষ্ট ধাপে ধাপে এগোনো।
🌍 বাস্তব উদাহরণ

Google Search Algorithm: তুমি কী লিখলে → সব website দেখলো → ranking করলো → সেরাটা দেখালো। এই পুরো process = Algorithm।

18

Logic

লজিক / যুক্তি ▼
📖
বাংলা অর্থ
সঠিক সিদ্ধান্ত নেওয়ার চিন্তার ধরন
💡
সহজ ব্যাখ্যা
Coding-এ logic মানে: "এই condition সত্য হলে এটা করো, নাহলে ওটা করো।" এটাই programming-এর brain।
🌍 বাস্তব উদাহরণ

ATM logic: Card correct? → PIN correct? → Balance আছে? → টাকা দাও। প্রতিটি "?" = একটি logical decision।

19

Syntax

সিনট্যাক্স ▼
📖
বাংলা অর্থ
Programming language-এর grammar বা লেখার নিয়ম
💡
সহজ ব্যাখ্যা
বাংলায় "আমি ভাত খাই" — এটা সঠিক syntax। "খাই ভাত আমি" লিখলে অর্থ হয় না। তেমনি code-এরও নিজের grammar আছে — সেটাই Syntax।
🌍 বাস্তব উদাহরণ

JavaScript-এ function লিখতে হয়: function myFunc() {"{}"} — এই নির্দিষ্ট নিয়মটাই Syntax।

20

Bug

বাগ / ত্রুটি ▼
📖
বাংলা অর্থ
Code-এ এমন ভুল যার কারণে program ঠিকভাবে কাজ করে না
💡
সহজ ব্যাখ্যা
পুরোনো দিনে computer-এ সত্যিকারের পোকা (bug) ঢুকে machine বন্ধ হয়ে গিয়েছিল। তখন থেকে code-এর ভুলকে Bug বলা হয়।
🌍 বাস্তব উদাহরণ

বকশী বাজারে ATM থেকে টাকা গেছে কিন্তু account-এ কাটেনি — এটা banking software-এর Bug!

21

Debugging

ডিবাগিং ▼
📖
বাংলা অর্থ
Code-এর Bug খুঁজে বের করে ঠিক করার প্রক্রিয়া
💡
সহজ ব্যাখ্যা
Programmer-দের কাজের ৫০% হলো Debugging। Code লেখার চেয়ে ভুল খুঁজে ঠিক করাতেই বেশি সময় যায়!
🌍 বাস্তব উদাহরণ

গাড়ি start নিচ্ছে না → mechanic খুঁজছে সমস্যা কোথায় → battery? fuel? engine? → খুঁজে ঠিক করলো = Debugging।

Section 2: Common Coding Terms

প্রোগ্রামিং-এ বারবার আসে এমন গুরুত্বপূর্ণ terms

▼
01

Variable

ভেরিয়েবল / চলক ▼
📖
অর্থ
Data সাময়িকভাবে রাখার জন্য নামযুক্ত container
⚙️
কাজ
যেকোনো value (সংখ্যা, text, তালিকা) store করে রাখা এবং পরে ব্যবহার করা।
💡 সহজ উদাহরণ

let name = "Aminul"; — এখানে name হলো Variable, "Aminul" হলো তার value। যেমন একটা বাক্স যার গায়ে "নাম" লেখা।

02

Constant

কনস্ট্যান্ট / স্থিরাঙ্ক ▼
📖
অর্থ
যে value একবার set করলে আর পরিবর্তন হয় না
⚙️
কাজ
Fixed value ধরে রাখা। যেমন PI = 3.14159 — এটা কখনো বদলাবে না।
💡 সহজ উদাহরণ

const MAX_USERS = 100; — এই 100 কখনো বদলাবে না। Variable-এর মতো, কিন্তু lock করা।

03

Function

ফাংশন ▼
📖
অর্থ
একটি নির্দিষ্ট কাজের জন্য তৈরি reusable code-এর block
⚙️
কাজ
একই code বারবার না লিখে, একবার function বানিয়ে রাখো — যতবার দরকার call করো।
💡 সহজ উদাহরণ

মিক্সার মেশিন = Function। যা দাও তা blend করে দেয়। তুমি প্রতিদিন মেশিন নতুন বানাও না — একবার কিনে বারবার ব্যবহার করো।

04

Parameter & Argument

প্যারামিটার ও আর্গুমেন্ট ▼
📖
অর্থ
Parameter = function-এর নির্ধারিত input slot। Argument = সেই slot-এ দেওয়া actual value।
⚙️
কাজ
Function-কে বিভিন্ন data দিয়ে আলাদা কাজ করানো যায়।
💡 সহজ উদাহরণ

মিক্সার-এ যে জায়গায় ফল দাও সেটা = Parameter। আজকে আম দিলে, কালকে কলা দিলে সেটা = Argument।

05

Array

অ্যারে / তালিকা ▼
📖
অর্থ
একসাথে অনেক value রাখার ordered তালিকা
⚙️
কাজ
একই ধরনের অনেক data একটি variable-এ রাখা।
💡 সহজ উদাহরণ

let fruits = ["আম", "কলা", "লিচু"]; — Shopping list = Array। প্রতিটি item একটি slot-এ থাকে।

06

Object

অবজেক্ট ▼
📖
অর্থ
Key-Value pair আকারে সংগঠিত data collection
⚙️
কাজ
Real-world জিনিসের properties code-এ represent করা।
💡 সহজ উদাহরণ

একজন মানুষ Object হলে: {"{"} name: "Aminul", age: 25, city: "Dhaka" {"}"} — ID card-এর মতো, একটা জিনিসের সব তথ্য এক জায়গায়।

07

Class

ক্লাস / নকশা ▼
📖
অর্থ
Object বানানোর blueprint বা ছাঁচ
⚙️
কাজ
একটা template বানিয়ে রাখো, সেই template থেকে অনেক Object তৈরি করো।
💡 সহজ উদাহরণ

Car-এর design = Class। সেই design থেকে লাল গাড়ি, নীল গাড়ি, বড় গাড়ি বানানো = Object। Factory = Class, প্রতিটি গাড়ি = Object।

08

Method

মেথড ▼
📖
অর্থ
Object বা Class-এর ভেতরে থাকা Function
⚙️
কাজ
Object-এর নিজস্ব action বা behavior define করে।
💡 সহজ উদাহরণ

গাড়ি Object-এর method হলো: .চালু_করো(), .থামো(), .হর্ন_দাও()। গাড়ির নিজের কাজ = Method।

09

Loop

লুপ / পুনরাবৃত্তি ▼
📖
অর্থ
একটি কাজ বারবার automatically করানোর technique
⚙️
কাজ
100 জনকে "Happy Birthday" message পাঠাতে 100 বার লিখতে হবে না — Loop একবার লিখলে বারবার করে।
💡 সহজ উদাহরণ

ফ্যান ঘোরে বারবার = Loop। কারখানায় machine একই কাজ হাজারবার করে = Loop। শর্ত শেষ না হওয়া পর্যন্ত চলতে থাকে।

10

Condition

কন্ডিশন / শর্ত ▼
📖
অর্থ
সত্য বা মিথ্যার উপর ভিত্তি করে সিদ্ধান্ত নেওয়া
⚙️
কাজ
if-else দিয়ে: "এই condition সত্য হলে A করো, নইলে B করো।"
💡 সহজ উদাহরণ

Traffic light: লাল হলে থামো, সবুজ হলে যাও। if (light === "red") stop(); else go(); — এটাই Condition।

11

Event

ইভেন্ট / ঘটনা ▼
📖
অর্থ
User বা system-এর কোনো action যা code trigger করে
⚙️
কাজ
Click, Scroll, Keyboard press, Form submit — সব Event। Event হলে specific code চলে।
💡 সহজ উদাহরণ

দরজার bell টিপলে ভেতরে sound হয়। Bell টেপা = Event, Sound = Response। Button click করলে popup আসা = Event handling।

12

Callback

কলব্যাক ▼
📖
অর্থ
একটি function যা অন্য function-এ pass করা হয় পরে চালানোর জন্য
⚙️
কাজ
কোনো কাজ শেষ হলে automatically পরবর্তী কাজ শুরু করে।
💡 সহজ উদাহরণ

মাকে বললে: "রান্না শেষ হলে আমাকে ডাকো।" সেই ডাক দেওয়াটা = Callback। কাজ শেষের পর automatically জানানো।

13

Promise

প্রমিস / প্রতিশ্রুতি ▼
📖
অর্থ
ভবিষ্যতে কোনো async কাজের result পাওয়ার প্রতিশ্রুতি
⚙️
কাজ
৩টি state আছে: Pending (অপেক্ষায়), Fulfilled (সফল), Rejected (ব্যর্থ)।
💡 সহজ উদাহরণ

Online order = Promise। Order দিলে = Pending। Delivery এলো = Fulfilled। Out of stock = Rejected।

14

Async / Await

অ্যাসিঙ্ক্রোনাস ▼
📖
অর্থ
একটি কাজ চলাকালীন অন্য কাজ করার পদ্ধতি — না থেমে
⚙️
কাজ
Server থেকে data আসার জন্য অপেক্ষা করতে গিয়ে পুরো page freeze না হওয়া।
💡 সহজ উদাহরণ

চা বানাতে দিলে, সেই ফাঁকে ফোন চেক করো — চা হলে পান করো। চায়ের জন্য বসে না থাকাটাই Async।

15

JSON

জেসন / ডেটা ফরম্যাট ▼
📖
অর্থ
JavaScript Object Notation — data আদান-প্রদানের standard format
⚙️
কাজ
Server ও Client-এর মধ্যে data পাঠানো-নেওয়ার জন্য universally ব্যবহৃত format।
💡 সহজ উদাহরণ

চিঠির envelope-এ নির্দিষ্ট নিয়মে ঠিকানা লিখতে হয় — সেটাই JSON-এর মতো। সব জায়গায় একই format, তাই সবাই বুঝতে পারে।

16

DOM

ডম ▼
📖
অর্থ
Document Object Model — webpage-এর সব element-এর tree structure
⚙️
কাজ
JavaScript DOM-এর মাধ্যমে webpage-এর যেকোনো element change, add, delete করতে পারে।
💡 সহজ উদাহরণ

পরিবারের গাছ chart কল্পনা করো: দাদু (html) → বাবা (body) → তুমি (div) → তোমার ভাই (p)। এই পুরো পারিবারিক tree = DOM।

Section 3: UI Components Dictionary

Web ও App-এর সব ধরনের visual elements

Dropdown

Navbar

Sidebar

Header

Footer

ফুটার / নিচের অংশ

Hero Section

হিরো সেকশন
📌কাজ: Website-এর প্রধান intro/banner অংশ
📍ব্যবহার: Landing page-এর প্রথম বড় section
✨এই page-এর উপরের বড় purple section = Hero Section
🃏

Card

কার্ড
📌কাজ: তথ্য সুন্দরভাবে box-এ সাজিয়ে দেখানো
📍ব্যবহার: Product list, Blog post, Profile preview
✨তোমার এই page-এর প্রতিটি term-box = Card
🪟

Modal

মোডাল / পপআপ উইন্ডো
📌কাজ: Background blur করে সামনে ভেসে ওঠা window
📍ব্যবহার: "Delete করবেন?" confirmation, Image preview
✨× close না করলে অন্য কিছু করা যায় না
💬

Popup

পপআপ
📌কাজ: হঠাৎ ভেসে ওঠা ছোট message বা window
📍ব্যবহার: Newsletter subscribe, Cookie consent, Promo
✨Modal-এর চেয়ে ছোট, page block করে না সবসময়
🪗

Accordion

একর্ডিয়ন
📌কাজ: Click করলে content খোলে/বন্ধ হয়
📍ব্যবহার: FAQ, Course section, Settings
✨এই page-এর প্রতিটি term card = Accordion!
🗂️

Tab

ট্যাব
📌কাজ: একই জায়গায় আলাদা content switch করে দেখায়
📍ব্যবহার: Browser tab, Profile (Posts/Photos/About)
✨ফাইল folder-এর ট্যাবের মতো
🎠

Carousel

ক্যারোসেল / স্লাইডার
📌কাজ: Image বা content auto-slide করে দেখায়
📍ব্যবহার: E-commerce banner, News headlines
✨Daraz-এর হোম page-এর উপরের ছবি sliding = Carousel
📝

Form

ফর্ম
📌কাজ: User-এর কাছ থেকে data collect করে
📍ব্যবহার: Login, Registration, Contact, Order
✨কাগজের ফর্ম পূরণের digital version
⌨️

Input Field

ইনপুট বক্স
📌কাজ: User type করে text বা number দিতে পারে
📍ব্যবহার: Search box, Username, Password
✨এই page-এর উপরের search bar = Input Field
☑️

Checkbox

চেকবক্স
📌কাজ: হ্যাঁ/না বা multiple option select করা
📍ব্যবহার: "Terms agree", To-do list tick, Filter
✨এই page-এ "পড়া হয়েছে" ✅ = Checkbox
🔘

Radio Button

রেডিও বাটন
📌কাজ: একাধিক option থেকে শুধু একটি select করা
📍ব্যবহার: Gender select, Payment method, MCQ quiz
✨Checkbox = একাধিক select। Radio = শুধু একটি।
🔄

Toggle Switch

টগল সুইচ
📌কাজ: On/Off switch — দুটি state-এর মধ্যে সুইচ
📍ব্যবহার: Dark mode, Notifications on/off, WiFi
✨এই page-এর 🌙 Dark mode button = Toggle Switch
🔍

Search Box

সার্চ বক্স
📌কাজ: Text type করে কিছু খোঁজার input field
📍ব্যবহার: Google search, Product search, User search
✨এই page-এর top search bar = Search Box
📄

Pagination

পেজিনেশন
📌কাজ: অনেক content-কে page-এ ভাগ করে দেখায়
📍ব্যবহার: Google search result, Product list
✨Google-এর নিচের 1 2 3 4 5 ... = Pagination
🍞

Breadcrumb

ব্রেডক্রাম্ব
📌কাজ: User এখন কোথায় আছে সেটা path দিয়ে দেখায়
📍ব্যবহার: E-commerce, Documentation site
✨Home → Electronics → Phones → iPhone = Breadcrumb
💭

Tooltip

টুলটিপ / ইঙ্গিত
📌কাজ: Hover করলে ছোট helper text দেখায়
📍ব্যবহার: Icon-এর উপর hover করলে কাজের বিবরণ

Badge

ব্যাজ / লেবেল

📍ব্যবহার: Notification count, "New" tag, "Hot" label
✨Facebook-এর bell icon-এ লাল ৩ = Badge
🍞

Toast / Notification

টোস্ট / নোটিফিকেশন
📌কাজ: কোনে ভেসে ওঠা temporary success/error message
📍ব্যবহার: "Saved!" ✅, "Error occurred" ❌
✨Toast pop up করে, কয়েক সেকেন্ড পরে মিলিয়ে যায়

Progress Bar

প্রগ্রেস বার

📍ব্যবহার: File download, Profile completion, Course progress
✨এই page-এর উপরের নীল bar = Progress Bar
⏳

Loading Spinner

লোডিং স্পিনার

📍ব্যবহার: Page load, Button click করার পর
✨YouTube video buffer হলে যে ঘুরন্ত চিহ্ন = Spinner

Skeleton Loader

স্কেলেটন লোডার
📌কাজ: Content load হওয়ার আগে content-এর outline দেখায়
📍ব্যবহার: Facebook feed, LinkedIn post loading
✨Spinner-এর চেয়ে বেশি professional — কঙ্কালের মতো shape দেখায়
📊

Dashboard

ড্যাশবোর্ড
📌কাজ: সব গুরুত্বপূর্ণ তথ্য একটি screen-এ overview
📍ব্যবহার: Admin panel, Analytics, Business report
✨গাড়ির সামনের speedometer + fuel gauge = Dashboard
🔍

কোনো result পাওয়া যায়নি। অন্য keyword দিয়ে চেষ্টা করুন।
